## Старт проекта

### Склонируй репозиторий и перейди в папку проекта
```
git clone git@github.com:CSSSR/csssr-project-template.git new-project && cd new-project
```

### Установи модули
```
npm i
```

### Запусти шаблон
```
npm start
```
