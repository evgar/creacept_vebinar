## Старт проекта

### Склонируй репозиторий и перейди в папку проекта
```
git clone https://github.com/evgar/creacept_vebinar.git creacept_vebinar && cd creacept_vebinar
```

### Установи модули
```
npm i
```
### или намного быстрее с помощью yarn
```
yarn i
```

### Запусти шаблон
```
npm start
```
